## What "10/10 working" means at this baseline

1. **Ship gates enforce safety automatically.** typecheck + lint +
   vitest + build + ratchets + block-audit + alert-coverage all run
   on every ship. Any regression fails loudly.

2. **Tenant isolation has tests at three layers.** Primitive
   (aging.test.ts), page (isolation.test.ts), API (the 4 fence
   tests). Clients cannot cross-tenant-read via URL manipulation,
   cookie forgery, or query-param injection.

3. **Silent failure classes are closed.** Rates refuse-to-calculate
   on stale config. Cron scripts alert on failure with structured
   logs. Forgeable cookies eliminated. system_config expiry gives
   7 days of heads-up.

4. **Documentation matches code.** HANDOFF for the week's state,
   CRON_MANIFEST for the 29 PM2 processes, MIGRATION_QUEUE for
   pending DB pushes, MAFESA_ONBOARDING for client #2 activation,
